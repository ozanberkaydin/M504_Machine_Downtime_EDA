Objective¶

The objective of this project is to analyze machine downtime patterns, identify operational issues, evaluate machine performance, and provide recommendations for improving manufacturing efficiency.

1. Business Context¶

Manufacturing companies depend on reliable machine performance to maintain productivity, product quality, and delivery targets. Unexpected machine downtime can disrupt production schedules, increase maintenance costs, and reduce overall equipment effectiveness.

This project analyses operational data collected from CNC machining equipment. The dataset contains machine identifiers, assembly line information, sensor measurements, and machine failure records. Key operational variables include hydraulic pressure, spindle speed, temperature, vibration, torque, and cutting force.

The objective of this analysis is to identify patterns associated with machine failures, evaluate machine performance, and generate data-driven recommendations for preventive maintenance. The findings can help maintenance teams reduce downtime, improve equipment reliability, and support operational decision-making.

2. Dataset Description¶

The dataset was obtained from Kaggle and contains CNC machine operating data with downtime status.

It has 2,500 observations and 16 variables covering machine ID, assembly line, date, sensor readings and downtime outcome.

Key fields include hydraulic, coolant and air pressure; coolant, hydraulic oil and spindle bearing temperature; spindle and tool vibration; spindle speed; voltage; torque; cutting force; and Downtime, which records Machine_Failure or No_Machine_Failure.

Because the dataset combines numerical sensor measures with categorical machine and line information, it is suitable for exploratory analysis of operating patterns linked to downtime.

Data Cleaning Summary¶

The initial checks found no duplicate records and several missing values across numerical sensor columns. Missing numerical values were replaced with the median of each column so that all 2,500 observations could be retained without giving excessive weight to extreme readings.

The Date column was converted to datetime format, and the failure outcome was mapped to a binary Failure_Flag for comparison, correlation and risk-matrix analysis.

Outliers were retained because unusual sensor values may represent abnormal machine behaviour. The pressure check found one negative hydraulic pressure value. It was kept as a possible sensor anomaly or abnormal condition, but hydraulic-pressure findings are interpreted with this data-quality point in mind.

After preprocessing, the cleaned dataset had 0 missing values and 0 duplicate records.

8. Final Discussion¶

This analysis used Python, Pandas and visualisation techniques to examine downtime patterns in CNC machine data. Grouped summaries, crosstabs, correlations, time analysis and a pressure-torque risk matrix were used to answer the business questions.

The strongest pattern is that machine failures are associated with lower hydraulic pressure and higher spindle speed. Vibration measures showed smaller differences, so they appear less useful as standalone indicators in this dataset.

The analysis has limitations. The dataset contains 2,500 records from three machines and does not include maintenance history, operators or production load. Some monthly and risk-matrix groups also contain fewer observations, so results should be treated as operational patterns rather than proof of causation.

For the business, the main action is to monitor hydraulic pressure and spindle speed more closely and prioritise inspection when machines enter higher-risk pressure and torque ranges.

9. Conclusion¶

This project identified operating factors associated with machine downtime in a manufacturing setting.

Hydraulic pressure was the clearest indicator in the dataset: lower pressure levels were linked with machine failures, while higher spindle speed also appeared during downtime events. Seasonal effects were weaker than operating-condition effects.

The pressure-torque matrix added useful context by showing that combined operating conditions can reveal higher-risk areas, although small cell counts should be considered when interpreting extreme percentages.

Overall, the analysis supports practical maintenance planning by highlighting hydraulic pressure and spindle speed as priority variables for monitoring and preventive action.
